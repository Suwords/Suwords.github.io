# 理想国的由来

>卷又卷不过，躺又躺不平，患得患失，有理想、迷茫，未知艺术家，小镇青年  
>深陷于代码的世界，寻一方角落，一张书桌，暖色的灯光，高垒的书籍  
>于深夜，在悠扬的音乐中编织自己的理想国  

困惑于25岁的人生，也于代码的世界里行走了7年，于数据的世界摸索了3年，深感这个世界的庞大，害怕在这个复杂、深邃的世界中迷失了方向，丢失了自我，谨以此站记录回忆、成长、蜕变以及心路历程、知识框架，同时希望能以自己为例，为同样迷茫的青年人、数据人、程序人提供一点点帮助，愿各位生而灿烂，如理想之国。
